ShareCertificate

The following variables are available. A difference is made between data valid for the year and data valid for a single address record and data valid for transactions for a adress record.

Annual data

  • ${year}: ${year}
  • ${plantPowerEarning}: ${plantPowerEarning}
  • ${printDate}: ${printDate}
  • ${signer1}: ${signer1}
  • ${signer2}: ${signer2}
  • ${totalNumberOfShareCertificate}: ${totalNumberOfShareCertificate}

Address record data

  • ${addressName}: ${addressName}
  • ${addressStreet}: ${addressStreet}
  • ${addressCity}: ${addressCity}
  • ${personalNumberOfShareCertificate}: ${personalNumberOfShareCertificate}
  • ${personalPowerEarning}: ${personalPowerEarning}
  • ${personalRemainingBalance}: ${personalRemainingBalance}

Transaction data

${transactionTable}: ${transactionTable}
